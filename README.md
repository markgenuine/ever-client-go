# go-ton-sdk

FreeTON SDK Golang language based itself on [TON-SDK](https://github.com/tonlabs/TON-SDK).

## Install
```sh
$ go get -u github.com/move-ton/go-ton-sdk
```
## Usage
```go
import goton "github.com/move-ton/go-ton-sdk"
```
